Percept – Accessibility Prompt Feedback Tool

Percept helps designers and developers understand how websites feel to people with disabilities and neurodivergent users. Select a profile, paste your HTML, and get feedback shaped by lived experience—not compliance checklists. Feedback comes in two flavors: emotional tone (how it feels) and technical guidance (how to fix it).

Optional: upload a profile JSON file in the same shape as profiles/*.json. This creates a temporary custom profile for the current session.

Paste HTML code (or plain text). The checker detects patterns that may cause barriers for the selected profile. Feedback tells you how it feels and how to improve it. Keyboard shortcuts: Ctrl/Cmd+Enter to analyze, Ctrl/Cmd+Shift+C to copy, Esc to clear.
