Percept – Accessibility Prompt Feedback Tool

Percept is built to help people with disabilities and neurodivergent users get clearer, more accurate responses from AI. It looks at the words you use in a prompt, points out where tone or structure might cause confusion and suggests ways to make the request easier for the AI to understand. The goal is to make sure the technology works for you, not against you.

Optional: upload a profile JSON file in the same shape as profiles/*.json. This creates a temporary custom profile for the current session.

Tip: You can paste raw HTML. The checker looks for keywords matched to the selected profile.
