Percept – Accessibility Prompt Feedback Tool

Percept helps people with disabilities and neurodivergent users get clearer, more accurate responses from AI. It reviews a prompt or HTML, identifies wording that may cause confusion, and suggests changes to improve clarity and tone. The tool supports profiles such as ADHD, dyslexia, screenreader, low vision, and motor disability, tailoring feedback to each.

Purpose and Audience
Percept is designed for anyone who wants AI output to be more usable and inclusive. It is aimed at people who face cognitive, sensory, or physical barriers that can be amplified by unclear prompts. The goal is to make the technology work for the user, not the other way around.

How It Works
Select a feedback style profile from the dropdown.
Paste HTML or write a plain text prompt into the input field.
Click "Analyze" to see feedback.
Review the tone preview and guidance, then make edits as needed.
Use "Copy Feedback" to copy the results to your clipboard.

Built With
HTML, CSS and vanilla JavaScript
Local JSON profiles for feedback rules
DOM manipulation for interactive UI
Keyword detection and tone previews
Grid-based layout with left input column and right feedback column
